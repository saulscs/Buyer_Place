# Buyer Place

Buyer Place is a market place design to connect people in a easy way to sell and buy  new or second hand ï£¿ Apple Products. you just need a couple of clicks to be in contact with a seller. On the other hand publish a product it so easy  too and fast.  


![](https://i.imgur.com/nJfFxdJ.png)


If you want taste my project just clone this repo and then:

* -In the backend folder install  `npm i` and run it with `npm run dev`
* In the frontend folder install `yarn install` and run it with `yarn start`

so now you can run it.
